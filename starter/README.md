# Starter Template

Questo starter è una struttura minimale per avviare rapidamente nuovi progetti Python/CLI.

- Use this template as a baseline for a new project: copy the template and customize.
- Contiene una piccola CLI skeleton, una gestione semplice delle dipendenze e una guida.

Come usarlo:
- Copia della cartella template in una nuova cartella progetto.
- Aggiorna i file (nome progetto, README, dipendenze).
- Installa le dipendenze: python -m venv venv; source venv/bin/activate; pip install -r requirements.txt
- Avvia o integra con il tuo flusso di lavoro.
